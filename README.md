# Capgemini Training – Java Full Stack

This repository is created to maintain my **daily practice and assessment work** during my **Java Full Stack training at Capgemini**.

All the code written during the training program is organized **day-wise** so that learning progress, concepts covered, and assessments can be easily tracked and reviewed.

---

## Purpose of this Repository

- To store daily Java practice programs
- To follow a clean and standard folder structure
- To maintain consistency throughout the training period
- To submit and showcase work for training assessments
- To track progress using GitHub

---

## Technologies Used

- Core Java  
- Java Development Kit (JDK)  
- Git and GitHub  
- Sublime Text  
- Command Line (Windows)  

---

## Coding and Repository Standards

- Only `.java` source files are committed  
- Compiled `.class` files are ignored using `.gitignore`  
- Clear folder naming for easy understanding  
- Meaningful commit messages  
- Code written as per training and assessment requirements  

---

## Daily Workflow

- Practice and write code locally  
- Organize files inside the respective `Day-XX` folder  
- Commit changes with a proper message  
- Push updates to GitHub daily  

---

## Training Status

This repository will be updated regularly throughout the Java Full Stack training period at Capgemini.  
It reflects continuous learning and hands-on practice as part of the training curriculum.
